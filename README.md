# driverless

Machine learning project for a driverless vehicle using TensorFlow

## WORK IN PROGRESS

## Setting up environment
Install [Anaconda](https://docs.continuum.io/)
```
conda env create -f environment.yml

activate tensorflow

python train_keras.py
```
